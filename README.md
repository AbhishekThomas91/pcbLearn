# 555 Timer STEM Toy PCB

## Overview

This project is a beginner PCB designed in KiCad as part of a STEM electronics learning exercise.

The circuit is based on the popular NE555 timer IC and demonstrates the complete PCB design workflow, including schematic capture, footprint assignment, PCB layout, routing, and 3D visualization.

## Features

* NE555 timer based circuit
* Through-hole components
* Beginner-friendly PCB design
* Designed using KiCad
* Suitable for STEM and electronics learning

## Components

| Component | Value                  |
| --------- | ---------------------- |
| U1        | NE555 Timer IC         |
| R1        | 10 kΩ Resistor         |
| R2        | 1 kΩ Resistor          |
| C1        | Capacitor              |
| C2        | Electrolytic Capacitor |
| TP1       | Battery Input          |
| TP2       | Human Contact Terminal |
| TP3       | Buzzer Output          |
